# Topic 5 Path Module

---

### **Concept**

**What is the Path Module?**

* The **Path module** is a **built-in Node.js core module** used for working with file and directory paths.
* It provides utilities to:

  * Build platform-independent file paths
  * Extract parts of paths (like directory name, base name, extension)
  * Normalize or join paths safely across OS (Windows/Linux/Mac)

---

### **Why Path Module?**

| Problem                               | Path Solution                          |
| ------------------------------------- | -------------------------------------- |
| OS-specific separators (`/` vs `\`)   | `path.join()` handles it automatically |
| Need to extract filename or extension | `path.basename()`, `path.extname()`    |
| Need full directory name              | `path.dirname()`                       |
| Need to create absolute paths         | `path.resolve()`                       |

---

### **How It Works**

* Node uses `path` to create and manipulate file paths **without worrying about the underlying OS format**.
* The module is **built-in**, so no installation required.
* You can import it using `const path = require('path');`

### **Code Example 1: Basic Path Operations**

**File:** `index.js`

```js
// ===========================
// Example: Basic Path Operations
// ===========================

// Import the built-in path module
const path = require('path');

// Get the current file name
console.log("File Name:", path.basename(__filename));

// Get the current directory name
console.log("Directory Name:", path.dirname(__filename));

// Get the file extension
console.log("File Extension:", path.extname(__filename));

// Parse the full path into an object
console.log("Path Object:", path.parse(__filename));

// Join paths safely (OS-independent)
const joinedPath = path.join(__dirname, 'data', 'sample.txt');
console.log("Joined Path:", joinedPath);

// Resolve an absolute path
const absolutePath = path.resolve('data', 'sample.txt');
console.log("Absolute Path:", absolutePath);
```

**Run in terminal:**

```bash
node index.js
```

**Output Example:**

```

File Extension: .js

  ext: '.js',
  name: 'index'
}
```

### **Code Example 2: Path Join vs Resolve**

**File:** `join-vs-resolve.js`

```js
// ===========================
// Example: Join vs Resolve
// ===========================

const path = require('path');

// path.join() simply joins parts together
const joined = path.join('/users', 'tanish', 'docs', 'file.txt');
console.log("Joined Path:", joined);

// path.resolve() starts from current working directory and builds absolute path
const resolved = path.resolve('users', 'tanish', 'docs', 'file.txt');
console.log("Resolved Path:", resolved);
```

**Output Example:**

```
Joined Path: \users\tanish\docs\file.txt
Resolved Path: C:\Projects\path-demo\users\tanish\docs\file.txt
```

### **Mini Project: File Path Utility**

**Goal:** Create a small Node.js script that analyzes a given file path and prints details.

**Folder:**

```
path-utility/
│
├── utils/
│   └── pathAnalyzer.js
└── app.js
```

**File:** `utils/pathAnalyzer.js`

```js
// ===========================
// Utility: Path Analyzer
// ===========================

const path = require('path');

function analyzeFilePath(filePath) {
  const details = {
    base: path.basename(filePath),
    dir: path.dirname(filePath),
    ext: path.extname(filePath),
    isAbsolute: path.isAbsolute(filePath),
    parsed: path.parse(filePath),
  };

  return details;
}

module.exports = analyzeFilePath;
```

**File:** `app.js`

```js
// ===========================
// Mini Project: Path Analyzer
// ===========================

const analyzeFilePath = require('./utils/pathAnalyzer');
const samplePath = 'C:/Users/Tanish/Desktop/NodeJS/path-demo/index.js';

const result = analyzeFilePath(samplePath);

console.log("File Path Details:\n", result);
```

**Run in terminal:**

```bash
node app.js
```

**Output Example:**

```
File Path Details:
{
  base: 'index.js',
  dir: 'C:/Users/Tanish/Desktop/NodeJS/path-demo',
  ext: '.js',
  isAbsolute: true,
  parsed: {
    root: 'C:/',
    dir: 'C:/Users/Tanish/Desktop/NodeJS/path-demo',
    base: 'index.js',
    ext: '.js',
    name: 'index'
  }
}
```

---

### **Notes**

* Use `path.join()` when you want to **concatenate paths safely**.
* Use `path.resolve()` to **get absolute paths**.
* `__dirname` gives **directory of the current file**, while `__filename` gives **the full file path**.
* Always use `path` module instead of manual `'/'` or `'\\'` concatenation to ensure **cross-platform compatibility**.
